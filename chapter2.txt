I learn about:
- cd ~: ma duce in C si in useru in care lucrez
- cd <file>: ma duce in ce fisier vreau 
- touch <file> : imi face un fisier
- start <file>: imi deschide un fisier
- ls : imi arata obiectele din fisier
- ls -a: imi arata fisierele ascunse 
- git init : imi face mediul git care este ascuns
- git status: imi arata ce fisiere sunt urmarite
- git add <file> : adauga fisier de uramrit si le pune in staging area
- git add . : pentru a adauga toate itemele din foldere
- git restore --staged <file>: pentru a nu mai urmari un anume fisier
- git restore --staged .: pentru a nu mai urmari toate fisierele
- git commit -m "descriere" : imi face o descriere la ceea ce fac, ar fi bine sa folosesc present tense
- git log: imi arata logourile
- git diff <file>: imi arata diferentetele produse fata de ultimul commit
- git checkout <file>:  imi da inapoi fisierul de la ultimul commit
